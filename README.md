# AI-Resume-Analyzer
This project is a rule-based intelligent Resume Analyzer developed in Java. It analyzes resume text files, extracts technical skills, calculates a resume score, and recommends suitable job roles.
📌 AI-Based Resume Analyzer & Job Recommendation System
🚀 Project Overview

This project is a rule-based intelligent Resume Analyzer developed in Java.
It analyzes resume text files, extracts technical skills, calculates a resume score, and recommends suitable job roles.

🔥 Features

Skill extraction using keyword matching

Resume scoring algorithm

Job role recommendation system

Suggestion engine for improvement

🛠️ Technologies Used

Java

OOP Concepts

File Handling

Rule-Based Logic

▶️ How to Run

Clone the repository

Open terminal

Compile:

javac Main.java

Run:

java Main

Enter resume file name (example: resume.txt)
